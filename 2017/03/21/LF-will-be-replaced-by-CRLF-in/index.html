LF will be replaced by CRLF in *

发表于 2017-03-21 | |

在使用git提交时出现错误提示

出现这个问题的原因是window平台下的换行符合Linux下的换行符不一致，Git是Linux平台下，所以在window平台下使用Git就会报这样的错.

解决方案

在shell中将autocrlf改为false。

接下来你就可以继续你的github之旅了。
